# Tablet-based-3D-App

Project for my Master's thesis on the topic "Tablet-based 3D Interaction".

# Description

Using multitouch and camera to interact with 3D objects. To track the users head OpenCV for Android is being used.
The Rajawali (OpenGL ES 2.0/3.0) engine takes care of the rendering part.

# Install

To use this project you have to download OpenCV for Android (Version 2.4.10) and import it as a module in Android Studio. A gradle build file should be generated automatically.
